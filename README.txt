SITE: Relatório / Portfólio - Esporte e Recreação da Unicap 2025

Como usar:
1. Extraia o arquivo ZIP.
2. Abra o arquivo index.html no navegador.
3. Para editar textos, abra index.html no VS Code.
4. Para editar visual/cores, abra style.css no VS Code.
5. As imagens ficam na pasta assets.

Arquivos:
- index.html
- style.css
- assets/ imagens usadas no relatório
